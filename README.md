# Amine Mahamat – Python Projects

Ce dépôt regroupe une sélection de mes meilleurs projets en Python afin de développer mes compétences en programmation, analyse de données et automatisation.

## Projets

### Analyse de ventes
Projet d’analyse de données utilisant la bibliothèque Pandas.  
Calcul du chiffre d’affaires, analyse par ville et par produit, et extraction de statistiques simples.

### Organisateur de fichiers
Script permettant de trier automatiquement des fichiers selon leur type (images, documents, code).  
Utilisation des modules `os` et `shutil` pour automatiser le classement.

### Analyse de données d’étudiants
Manipulation d’un dataset pour calculer des moyennes, comparer les résultats et identifier les meilleures performances.

### Système de recommandation
Implémentation simple d’un système de recommandation basé sur le filtrage par genre avec interaction utilisateur.

### Gestionnaire d’étudiants
Programme en console permettant d’ajouter, afficher et rechercher des étudiants.  
Travail sur les structures de données et la logique de programmation.
